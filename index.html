I wanted to thank you for the support given to EnterIN.

EnterIN and his "Inception" effect has astonished the whole world, and for this reason I inform you about the birth of a new reality!

EnterIN is going to become a Frontend Framework that will enable all the web developers to create a real world inception into their web pages!
This framework will be distributed completely Open Source to the whole world, licensed under the AGPL3 license.

What will this new framework do?

I'm making a simple framework called EnterIN and unique respecting his "Inception Logic".
Things that are planned for future releases:
  • Quick Configuration and User Friendly
  • Using the latest HTML5 / CSS3 technologies
  • Make it fully compatible with CMS out already, like Wordpress but not limiting to it
  • Touch / Mobile compatibility with iOS , Android , Symbian and Windows Phone
  • 3D Image Effects without WebGL
  • Compatible with events with keyboard and mouse wheel

What is an "Inception Effect"?

EnterIN has its own logic for page view , and it is the same logic from where the jQuery plugin has been borned, which gave birth to the world's EnterIN visits.
The "Inception Effect" is the power of EnterIN!

Why "not using WebGL 3D effects"?

EnterIN wants to enclose all the resources which are present on the web, but with its own innovation.
The Canvas and the WebGL technologies would penalize SEO, and this the reason I am working hard to simulate 3D effects with maximum performances!

Why the AGPL 3 license?

EnterIN isn't only a framework: it's a Social Project.
I believe in innovation, development, collaboration, and world sharing, and this the reason of why I choose to release EnterIN for Free and as an Open Source project with an AGPL3 license.
Anyone can download, use, and sell it, but the framework can't stop its "cycle" of diffusion and can't therefore be enclosed in a Proprietary Software under another license .
Any changes and any plugin created with EnterIN, must be released under the same license.
The development and innovation should not be stopped. This is the philosophy of EnterIN!
